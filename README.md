# Redesigned_Youtube
It's a re-designed YouTube with maybe with better UX, functionality and aesthetic.


The core idea behind the UI is extremely simple: on the left-hand side, we would have the context of why we are. This can be like YouTube, Gemina, or whatever web page or app we are in. We will see the name of it, or it can also be implemented in a way like maybe you are in a text editor and the file name or the text file's name. You can put it there. It is basically on the left side that we give the user the context of, "Hey, this app is about this." This is really important because our eye actually starts from there, and telling the user what's where they are is actually way more important and in the middle, we have the Navigation. Basically, it's from Navigation, and on the right-hand side we put the actions. In the middle, we would have Navigation, and for some special function, maybe search or maybe that, we can implement it in a way that just looks nice. The idea of the whole thing is actually very simple and concise.

On the right-hand side, we put the action, and on the left-hand side we put the context. In the middle, we put the navigation. Everything is on the top, so you never get lost or anything.
